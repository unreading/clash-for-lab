## 项目简介

基于 [clash-for-linux-install](https://github.com/nelvko/clash-for-linux-install) 项目进行二次开发的[clash-for-lab](https://github.com/SaladDay/clash-for-lab)进行的三次开发。

在 clash-for-lab 的基础上:
1. 增加了自定义安装目录
2. 增加了一些命令，主要包括
   - 可以查看当前节点延时的 mihomo now
   - 可以管理多个订阅的 mihomo subscribe/sub
   - 可以更改订阅和节点和代理模式的mi ch
3. 重新修改了一下mi ui 的信息提示以及web控制台的更改。
